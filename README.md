# Dorsal Horn Synapse Mapper

A drag-and-drop browser game for PT pain science students. Learners place neurotransmitters and inflammatory mediators onto a clean anatomical diagram of peripheral nociceptor input and dorsal horn modulation.

## Learning Targets

- Peripheral nociceptor sensitization in skin
- Primary nociceptive afferent synapse onto a second-order dorsal horn neuron
- Inhibitory interneuron effects on first-order and second-order neurons
- Descending modulation from RVM and LC inputs
- Ascending pain pathway direction toward the thalamus

## Included Mediators

- Histamine
- Cytokines
- Prostaglandins
- Glutamate
- Substance P
- GABA
- Glycine
- Enkephalin
- Serotonin
- Norepinephrine

## Run Locally

Open `index.html` in a browser, or serve the folder with any static file server.

No build step or external dependency is required.
